# Third-party notices

The core `@flyfish-dev/html2pdf-js` package in this version has no runtime third-party rendering dependency. It does not import IronPress, html2canvas, jsPDF, Paged.js, Puppeteer, or a browser print adapter.

The demo application still uses Vite and TypeScript as development/build tooling. Review `package.json` and `pnpm-lock.yaml` for development dependency versions.
